# Gizmo Galaxy Backend

## Overview
The Gizmo Galaxy Backend is a Java Spring Boot application designed to support an e-commerce platform. It provides the necessary APIs for managing products, orders, users, and payment processing.

## Features
- **Product Management**: CRUD operations for products, including categories, prices, and inventory.
- **User Authentication**: Secure user registration, login, and account management.
- **RESTful APIs**: Well-defined endpoints to allow easy consumption from frontend applications.
- **Microservices Ready**: The architecture allows for scalability and integration with other services.

## Technologies Used
- **Java**: Main programming language for the application.
- **Spring Boot**: Framework for building the application swiftly and efficiently.
- **Spring Security**: To handle authentication and authorization.
- **JPA/Hibernate**: For database interactions and ORM.
- **MySQL/PostgreSQL**: Relational database management system.
- **Maven/Gradle**: Build tools for dependency management.

## Getting Started
### Prerequisites
- JDK 11 or later
- Maven or Gradle
- MySQL/PostgreSQL installed

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ChinmayRaj/GizmoGalaxy_Backend.git
   cd GizmoGalaxy_Backend
Set up your database and update the application.properties with your database credentials.
Run the application:
Using Maven: mvn spring-boot:run
Using Gradle: ./gradlew bootRun
API Documentation
Refer to the Postman Collection for API endpoints and usage.
